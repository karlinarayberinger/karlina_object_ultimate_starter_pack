MULTIVERSE

The following terms and their respective definitions (and elaborating paragraphs) describe NATURE as being a static “block” which contains all decision-making trajectories.

STATIC: an adjective which describes some partition of nature as not changing according to some frame of reference observing that partition of nature.

CHANGE: a noun which describes some frame of reference observing a particular phenomenon either emerging out of pure nothingness or else disappearing into pure nothingness.

In order for change to be observed, the same allocation of space being observed must be observed for some interval of time whose length is a nonzero quantity.

It may be inferred that change, and therefore the passage of time, is perceptible from a partial frame of reference but not from an omniscient frame of reference because an omniscient frame of reference would observe all phenomena as occurring simultaneously and, hence, timelessly. Therefore, in order for an omniscient frame of reference to perceive change (and, hence, experience the passage of time), that omniscient frame of reference must “zoom into” exactly one of many partial frames of reference and trade omniscience for partiality.

SPACE_TIME_CONTINUUM: a linear and unidirectional succession of some natural number of (or else infinitely many) timeless snapshots of apparently one finite allocation of space.

A phenomenon occurring within a particular space-time continuum which does not appear to change as time elapses within that space-time continuum is considered to be a static phenomenon (according to the frame of reference which renders that space-time continuum).

UNIVERSE: exactly one space-time continuum which exactly one frame of reference appears to itself to be inhabiting (and which that frame of reference is rendering).

According to the frame of reference which is rendering a particular universe, that frame of reference is static while the phenomena which comprises that universe is changing.

If more than one universe exists, then a frame of reference which renders exactly one universe while not rendering any other universes is considered to be partial rather than omniscient.

If more than one universe exists, then a frame of reference which simultaneously renders each one of those universes is considered to be omniscient rather than partial.

If an information processing agent (which is the source of exactly one frame of reference per timeless instant along exactly one corresponding space-time continuum) named K perceives its environment as containing more than one information processing agent, then every information processing agent which is not K and which K perceives as existing in the same universe as K is merely a phenomenon (according to K) occurring inside of K’s frame of reference regardless of whether or not any of those other information processing agents inhabits (and renders) its own solipsistic universe. While K is confined to its own frame of reference, K cannot determine whether or not any other frames of reference exist. If each of those information processing agents which K observes as being external to K projects its own uniquely corresponding frame of reference, then the universe in which K inhabits is a compound universe (i.e. a universe which contains multiple universes inside of it and vice versa).

MULTIVERSE: the set which contains all universes.

It may be inferred that there is exactly one immutable multiverse (and that multiverse may contain zero universes, some natural number of universes, or else indefinitely many universes).

If the words ensconced inside of this web page are currently being observed by any frame of reference, then at least one universe exists (and the words ensconced inside of this web page are composed of space, time, matter, and energy rather than being composed exclusively of pure nothingness).

MANY_WORLDS_HYPOTHESIS: the theory that more than one universe exists and that one universe branches into multiple phenomenally distinct universes whenever a decision is made inside of a parent universe.

A decision is a process which enables change to occur. From the context of a computer program, a decision is made when some information processing agent receives some finite piece of informational input and uses that informational input to determine which finite piece of informational output from a list of multiple informational outputs to select as the one and only output. For example, if a computer program receives an input named X and that input represents the integer 2, then, if the decision making algorithm is a function which takes any X as an input and which yields exactly one logically corresponding output named Y (and that function is Y := f(X) = X + 1), then the outcome to that decision is 3 (and not any other piece of information such as 4 or 5).

The following hypothetical scenario illustrates what the many worlds hypothesis models: Suppose some information processing agent is traveling through a maze and reaches a point in the maze where one path branches into two distinct paths. Suppose then that the information processing agent uses an algorithmic process of elimination to decide which one of those two paths is most useful to that information processing agent’s goals to traverse. If the information processing agent decides to traverse the left path, then, according to the many world’s hypothesis, in some parallel universe, a parallel version of that information processing agent has chosen to traverse the right path instead of the left path.

According to some variations of the many worlds hypothesis, the total number of parallel universes which stem from a common origin universe after an information processing agent inhabiting (or, more accurately, projecting) that origin universe makes a decision is the number of options which that information processing agent apparently could have selected exactly one of during that decision-making process.
