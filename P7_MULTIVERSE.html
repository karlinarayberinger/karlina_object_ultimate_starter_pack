MULTIVERSE

The following terms and their respective definitions (and elaborating paragraphs) describe nature as being a static “block” which contains all decision-making trajectories.

STATIC: an adjective which describes some partition of nature as not changing according to some frame of reference observing that partition of nature.

CHANGE: a noun which describes some frame of reference observing a particular phenomenon either emerging out of pure nothingness or else disappearing into pure nothingness.

In order for change to be observed, the same allocation of space being observed must be observed for some nonzero interval of time.

It may be inferred that change, and therefore the passage of time, is perceptible from a partial frame of reference but not from an omniscient frame of reference (because an omniscient frame of reference would observe all phenomena as occurring simultaneously and therefore timelessly).

SPACE_TIME_CONTINUUM: a linear succession of some natural number of (or else infinitely many) timeless snapshots of apparently one finite allocation of space.

A phenomenon occurring within a particular space-time continuum which does not appear to change as time elapses within that space-time continuum is considered to be a static phenomenon (according to the frame of reference which renders that space-time continuum).

UNIVERSE: exactly one space-time continuum which occurs inside of exactly one frame of reference.

If more than one universe exists, then a frame of reference which renders exactly one universe is considered to be partial rather than omniscient because all other universes are excluded from that frame of reference.

If more than one universe exists, then a frame of reference which simultaneously renders each one of those universes is considered to be omniscient rather than partial.

MULTIVERSE: the set which contains all universes.

It may be inferred that there is exactly one immutable multiverse (and that multiverse may contain zero universes, some natural number of universes, or else indefinitely many universes).

If the words ensconced inside of this web page are currently being observed by any frame of reference, then at least one universe exists.

MANY_WORLDS_HYPOTHESIS: the theory that more than one universe exists and that one universe branches into multiple phenomenally distinct universes whenever a decision is made inside of a parent universe.

Suppose some information processing agent is traveling through a maze and reaches a point in the maze where one path branches into two distinct paths. Suppose then that the information processing agent uses an algorithmic process of elimination to decide which one of those two paths is most useful to that information processing agent’s goals to traverse. If the information processing agent decides to traverse the left path, then, according to the many world’s hypothesis, in some parallel universe, a parallel version of that information processing agent has chosen to traverse the right path instead of the left path.
