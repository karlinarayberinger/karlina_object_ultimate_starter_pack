The following paragraphs describe the goals of the author of the website whose Uniform Resource Locator is phonetically worded as Karlina Object dot WordPress dot Com.

Note that the term ethics refers to the conceptualization and the implementation of goal-oriented behaviors.

The purpose of this website is to preserve knowledge which its one and only author (i.e. Karlina Ray Beringer) wants to remember.

One of the goals which the author of this website has is to extend the lifetime of the information processing agent which is the author of this website for as long of a time period as possible and, also, to extend the existence of the intellectual property which comprises this website (and websites which this website links to and which the author of this website also authored) for as long of a time period as possible.

One hypothesis which the author of this website has is that it is possible for the intellectual property which comprises this website to be transcribed from one substrate to another substrate in a verbatim retrievable format such the same information processing agent can access that same transcribed intellectual property infinitely many times (which would imply that the information processing agent which is observing each one of those transcriptions experiences a continuity of consciousness which is infinitely long). In other words, such an attention span would be immortal yet still experience the passage of time.

IDEOLOGY: an information filtering framework which is programmed into an information processing agent and which classifies input or throughput data in terms of how useful that data is within the context of achieving the goals which that information processing agent has.

(For example, an information processing agent whose only goal is collecting as many gems from a particular mine as possible before that mine closes would likely classify information which is not related to that goal as irrelevant while paying attention to and remembering information about where gems have been reported by other miners to be most abundant within that mine).

HEDONISM: a rudimentary ideology whose sole value is the avoidance of suffering.

(The author of this website defines suffering as an information processing agent experiencing itself being deprived of what that information processing agent considers to be at least one of its highest priority values).

NIHILISM: the absence of an ideology; a lack of preferences; a lack of there being any sense of progress due to the fact that there are no goals to attain and no values by which to organize one’s behavior and thought processes.

VALUE: an abstract sense of what an information processing agent thinks is ideal (i.e. desirable) for that information processing agent to experience in its future and to continuously seek to maximize the experience of.

GOAL: specific imaginary future conditions which instantiate the values which an information processing agent has and which organize that information processing agent’s cognitive resources towards attaining.
