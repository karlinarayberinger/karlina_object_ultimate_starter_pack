The following terms and their respective definitions (and elaborating paragraphs) describe the intellectual property which comprises this website (i.e. the website whose domain name is phonetically worded as Karlina Object dot WordPress dot Com).

FILE: a named sequence comprised of some natural number of binary digits which are arranged in exactly one order and which represents a particular piece of information which is verbatim transmissible as a sequence of ones and zeros.

An example of a file is the C++ source code file named factorial.cpp (which is used to compute N factorial (assuming that N is a sufficiently small natural number) by multiplying N by exactly one instance of each unique natural number which is less than N in the computer programming tutorial web page of this website named FACTORIAL).

INTELLECTUAL_PROPERTY: a collection of some natural number of files.

OPEN_SOURCE: an adjective which describes some piece of intellectual property as being legal for any person to make indefinitely many copies of, to distribute any of those copies (either freely or in exchange for money, valued products, or valued services) to any recipient, to include wholly or partially as part of one’s own intellectual property, or to claim as one’s own intellectual property.

Note that, if a person makes a copy of some piece of intellectual property which is licensed by that intellectual property’s original creator as being open source, then that copy is treated as its own separate piece of intellectual property (even though the source code which comprises the copy is identical to the source code which comprises the original). For example, if Person A publishes a piece of intellectual property named A.cpp and licenses that file as being open source, then Person B can legally make a copy of that file and either use the same file name for the copy file or else give that copy file a different name such as B.cpp and either license that copy file as open source or else as some proprietary piece of intellectual property. (Person B’s copy of A.cpp does not legally change any of the attributes of Person A’s original A.cpp file (including the fact that Person A’s original A.cpp file is licensed as open source)).

Within the context of this website, the terms open source and public domain are used interchangeably. Public domain generally refers to ownerless intellectual property (and if some piece of intellectual property is labeled as being part of the public domain, then that piece of intellectual property can be legally treated as any person’s intellectual property).

Note that the intellectual property which comprises each web page within this website (i.e. the website whose domain name is phonetically worded as Karlina Object dot WordPress dot Com) is licensed by the author of this website (i.e. Karlina Ray Beringer) as being open source.

Backed up versions of each one of the web pages in this website are available to view at Archive dot Org (and were saved to Archive dot Org using the WayBack Machine).

PROPRIETARY: an adjective which describes some piece of intellectual property as being illegal to use or to make copies of without permission from that intellectual property’s owner.

The owner(s) of some proprietary piece of intellectual property may or may not be the original creators of that piece of intellectual property. For example, if Person A publishes a piece of intellectual property named A.cpp and licenses that file as being open source, then Person B can legally make a copy of that file and license that copy file as being Person B’s own proprietary intellectual property (but Person B cannot legally remove the open source license from Person A’s A.cpp file).

A person might want to license a piece of intellectual property as proprietary rather than open source in order to ensure that the person gets paid each time a customer downloads a copy of that intellectual property (and to prevent whoever downloads a copy of that intellectual property from making copies of that copy).
